## Short Description (132 characters max)

## Detailed Description (For Chrome Web Store)

### Transform Your Credentialing Workflow

**TodaysDental Credentialing Autofill** is the ultimate Chrome extension for dental practices, enabling instant, accurate form filling across 1,800+ insurance provider portals.

### 🚀 Key Features

**Smart Portal Detection**
Automatically detects supported insurance portals including CAQH ProView, Delta Dental, Cigna, Aetna, UnitedHealthcare, Blue Cross Blue Shield, Humana, MetLife, Guardian, and 1,800+ more.

**One-Click Autofill**
Select a provider from your TodaysDental database and fill entire credentialing forms with a single click. No more copy-pasting from spreadsheets!

**Pre-Fill Review**
Review all data before it's entered into forms. See exactly what will be filled and make adjustments as needed.

**Field Highlighting**
Visual indicators show which fields will be filled, which are empty, and which may have conflicts.

**Secure & Private**
All data is encrypted in transit. Your provider information never leaves your control.

### 📋 Supported Portals (1,800+)

- **Clearinghouses**: CAQH ProView, Availity, Navinet
- **Major Dental**: Delta Dental (all states), Cigna Dental, MetLife Dental, Guardian Dental
- **Health Insurers**: Aetna, Anthem, UnitedHealthcare, Blue Cross Blue Shield (all states)
- **Managed Care**: Humana, WellCare, Molina, Centene
- **Regional Plans**: Kaiser Permanente, Regence, Priority Health, UPMC
- **Medicaid/Government**: All state Medicaid portals, TRICARE
- **Specialty**: Liberty Dental, DentaQuest, MCNA, Dentegra

### 🔒 Security First

- TLS 1.3 encrypted connections
- No data stored locally
- Audit trail for compliance
- SSO integration with TodaysDental

### 💡 How It Works

1. **Install** the extension from Chrome Web Store
2. **Sign in** with your TodaysDental credentials
3. **Navigate** to any insurance portal
4. **Select** your provider from the dropdown
5. **Click** "Autofill" and watch the magic happen!
